DedispersionConfig¶

class pirate_frb.DedispersionConfig¶

Configuration for dedispersion processing.

Specifies frequency channels, time samples, dedispersion tree parameters, downsampling levels, peak-finding configuration, early triggers, and GPU settings. Can be loaded from YAML files or constructed programmatically.

See configs/dedispersion/chord_sb2_et.yml for an example with per-field documentation.

Example usage:

# Load from YAML
config = DedispersionConfig.from_yaml('config.yaml')

# Create and configure programmatically
config = DedispersionConfig()
config.zone_nfreq = [1024]
config.zone_freq_edges = [400.0, 800.0]
config.time_sample_ms = 0.983
config.tree_rank = 13

Early triggers are automatically kept sorted by ds_level (increasing) and delta_rank (decreasing).

Parameters:
