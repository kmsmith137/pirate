# gRPC Protocol Definitions

The FRB search engine defines three gRPC services, one per `.proto` file:

- [frb_search.proto](frb_search.proto) -- the `FrbSearch` service (FRB server RPC interface)
- [frb_grouper.proto](frb_grouper.proto) -- the `FrbGrouper` service
- [frb_sifter.proto](frb_sifter.proto) -- the `FrbSifter` service
